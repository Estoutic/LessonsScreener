# Screener

Chrome-расширение для автоматического сохранения планов уроков с kiber-one.pro в PDF.

## Установка

### Из ZIP-архива

1. Распаковать `screener-extension.zip`
2. Открыть `chrome://extensions/`
3. Включить **Developer mode** (переключатель справа вверху)
4. Нажать **Load unpacked**
5. Выбрать распакованную папку `dist/`

### Из исходников

```bash
cd extension
npm install
npm run build
```

Затем загрузить папку `extension/dist/` через Chrome (шаги 2–5 выше).

## Как пользоваться

1. Открыть kiber-one.pro и залогиниться
2. Перейти на страницу модуля с уроками (где видны PDF-превью)
3. Нажать на иконку расширения **Screener** в панели Chrome
4. Нажать **Scan** — расширение найдёт все уроки на странице
5. Выбрать урок в выпадающем списке (или оставить «All» для всех)
6. Нажать **Start**

Расширение автоматически:
- перелистывает страницы каждого урока
- делает скриншот каждой страницы
- собирает все скриншоты в PDF
- сохраняет PDF в `~/Downloads/lessons/`

Имя файла берётся из заголовка урока на сайте (например, `Клавиатурный тренажёр. Урок 1.pdf`).

### Кнопки

| Кнопка | Что делает |
|--------|-----------|
| **Scan** | Сканирует страницу, находит уроки и заполняет выпадающий список |
| **Start** | Запускает захват выбранного урока (или всех) |
| **Stop** | Останавливает захват |
| **Test** | Делает один тестовый скриншот текущей страницы |

### Статусы в попапе

| Поле | Значение |
|------|---------|
| **Status** | idle / capturing / done / error |
| **Current page** | Номер текущей страницы |
| **Captured** | Сколько скриншотов уже сделано |

## Куда сохраняются файлы

- Скриншоты страниц: `~/Downloads/screener/page-001.png`, `page-002.png`, ...
- Готовые PDF: `~/Downloads/lessons/Название урока.pdf`
